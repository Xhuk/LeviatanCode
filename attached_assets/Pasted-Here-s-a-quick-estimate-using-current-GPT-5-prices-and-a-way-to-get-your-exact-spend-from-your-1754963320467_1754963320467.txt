Cost formula you can trust:
`cost = (input_tokens/1M × price_in) + (output_tokens/1M × price_out)` — for GPT-5 / mini / nano the current prices are **\$1.25 / \$10.00**, **\$0.25 / \$2.00**, **\$0.05 / \$0.40** per million tokens. ([OpenAI][1], [Simon Willison’s Weblog][2], [TechCrunch][3])

Want your **actual** spend? Log usage from the API response and total it:

```js
// after openai.responses.create(...)
const u = response.usage; // {input_tokens, output_tokens, reasoning_tokens?}
console.log("usage", u);
// estimate cost (USD):
const PRICES = {
  "gpt-5":      { in: 1.25, out: 10.0 },
  "gpt-5-mini": { in: 0.25, out: 2.0 },
  "gpt-5-nano": { in: 0.05, out: 0.4 },
};
function estimateUSD(model, usage){
  return (usage.input_tokens/1e6)*PRICES[model].in +
         (usage.output_tokens/1e6)*PRICES[model].out;
}
```

The Responses API includes a `usage` object in the result (you can also see token counts in the dashboard/docs). Store each call’s `usage` and model in `localStorage` or your backend, then sum with the formula above for a precise monthly total.
